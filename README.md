**估值不能指导短期走势（09:27:35）**

昨天有朋友留言，说哎呀如果我重视你说的环保95%分位，卖掉就好了，可以躲过大跌。



我请所有朋友一定要清楚明白一件事：



贵，不代表短期一定跌。便宜，也不代表一定会涨。这句话非常非常非常重要，请一定要铭记在心。



如果估值不能指导短期走势，那么我用十几年的时间去统计、记录、分析它又有什么用呢。



之前我举过一个例子。就是大家可以把在股市投资想象成吃酒席。



在酒席上，有人坐那等了半天，刚开席吃了两口就跑了。有人开席后坐下吃，买单前跑掉了。有人宴席快结束了坐下了，刚喝了两口肉汤，服务员来了：



“先生，您这桌一共消费*******元，请您买单。”



哥们儿抬头一看，一桌就剩自己了。



我们研究一个品种贵还是便宜，类似于你掌握了这张宴席的菜单，以及上菜顺序。你能模模糊糊知道大概什么时候坐下，未来会有宴席吃。也能大概知道上完哪道菜离服务员过来收钱就不远了。但是，你不能确切的知道他什么时候来，所以你需要选择，是再多吃几口，还是赶紧走人。



有些人不贪，吃饱了就行，知道硬菜上的差不多了，起身走了。有些人走了以后一转身趴在门缝看，看服务员是不是又上了什么硬菜。如果真的有加菜，他们会捶胸顿足，埋怨自己走的太早了，这么好吃的菜没吃到。



当市场疯狂的时候，有可能不断的加菜，这时候，理性离席的人，能把持住自己吗。



那么我怎么应对这个问题。



我的做法是，研究上菜单，觉得正餐上的差不多了，就往门口的座位挪一挪。门口的座位可能会少吃几口菜，但安全多了。然后注意听门外服务员的脚步声。一旦听到服务员过来，马上起身离开，争取不买单。



往门口挪，就是左侧操作慢慢卖出，少吃点菜。听脚步，就是右侧，该跑就跑。



估值就是上菜单，能告诉你菜上的怎么样了。但不能告诉你有没有加菜，什么时候服务员过来收钱。

对于个人的自由现金流如何理解呢。



我认为，对于个人与家庭的自由现金流，就是你的收入（正业+副业+投资收益流入）- 生活支出（日常开销+租房或自住房月供）- 资本支出（投资房月供+金融投资投入）= 个人/家庭自由现金流。

如果你的年度正业收入是100,000元，副业收入是30,000元，股息、出租房屋、基金分红、利息、理财分红是20,000元，那么，你的收入项就是150,000元。
