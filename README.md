**做人的底线（13:53:56）**

**估值不能指导短期走势（09:27:35）**

昨天有朋友留言，说哎呀如果我重视你说的环保95%分位，卖掉就好了，可以躲过大跌。



我请所有朋友一定要清楚明白一件事：



贵，不代表短期一定跌。便宜，也不代表一定会涨。这句话非常非常非常重要，请一定要铭记在心。



如果估值不能指导短期走势，那么我用十几年的时间去统计、记录、分析它又有什么用呢。



之前我举过一个例子。就是大家可以把在股市投资想象成吃酒席。



在酒席上，有人坐那等了半天，刚开席吃了两口就跑了。有人开席后坐下吃，买单前跑掉了。有人宴席快结束了坐下了，刚喝了两口肉汤，服务员来了：



“先生，您这桌一共消费*******元，请您买单。”



哥们儿抬头一看，一桌就剩自己了。



我们研究一个品种贵还是便宜，类似于你掌握了这张宴席的菜单，以及上菜顺序。你能模模糊糊知道大概什么时候坐下，未来会有宴席吃。也能大概知道上完哪道菜离服务员过来收钱就不远了。但是，你不能确切的知道他什么时候来，所以你需要选择，是再多吃几口，还是赶紧走人。



有些人不贪，吃饱了就行，知道硬菜上的差不多了，起身走了。有些人走了以后一转身趴在门缝看，看服务员是不是又上了什么硬菜。如果真的有加菜，他们会捶胸顿足，埋怨自己走的太早了，这么好吃的菜没吃到。



当市场疯狂的时候，有可能不断的加菜，这时候，理性离席的人，能把持住自己吗。



那么我怎么应对这个问题。



我的做法是，研究上菜单，觉得正餐上的差不多了，就往门口的座位挪一挪。门口的座位可能会少吃几口菜，但安全多了。然后注意听门外服务员的脚步声。一旦听到服务员过来，马上起身离开，争取不买单。



往门口挪，就是左侧操作慢慢卖出，少吃点菜。听脚步，就是右侧，该跑就跑。



估值就是上菜单，能告诉你菜上的怎么样了。但不能告诉你有没有加菜，什么时候服务员过来收钱。

**真正的高标准严要求（09:28:20）**

昨天有朋友在留言里说：你大跌之前卖一份50，那剩下的149份呢。


言下之意，无非是卖少了。

然后他又说，5月卖出的中证500怎么说？


言下之意，就是中证500卖早了，吃少了。


我们不聊500本来就是波段仓位，明明白白写清楚的“波段”。我们来分析一下，这位朋友对别人的要求。


他的两句话，代表了他对别人的两个要求，什么要求呢：


第一，高位卖出要大仓位卖出。卖少了不算。第二，低位绝对不能卖出，即使你说清楚了就是吃波段。你卖了就是错，哪怕只卖了一份波段仓位，这一份波段仓位十几天赚了十几个点也不行。


如果你的逻辑思维是完整的，你能不能看出这两者的自相矛盾？


这意味着什么呢，意味着他要求别人，要完全把握市场行情波动，一点点都不能错。如果你低位吃波段卖了一点点点，那就是错。如果你高位没有大仓位卖出，那也是错。


反过来，如果5月不是低位，而是一个高位，那即使你卖了也是错。为什么呢，因为你只卖了一点点，为什么没有大仓位卖出？如果前几天50不是高位，而是继续涨，那也是错，因为你卖了，虽然你只卖了1份。


也就是说，从他的逻辑看，无论你怎么做，都是错。无论怎么做，全是错。

**你能学到什么（09:53:07）**

最近几个月被刷屏的银行事件，有了一些进展。5万以下的账户有人管了。


我知道也有一些关注我的朋友碰到了这件恶心事，衷心希望你们能顺利取回自己的钱。


各位围观的朋友，我不知道各位是否在看热闹或者为当事群众郁闷的同时，有没有学到什么呢。

第二件事，是任何超出同类水平过多的金融产品，其风险等级也会几何级的上升。比如说，5%的“无风险产品”，其风险等级要高于3%的“无风险产品”非常非常多。有人说5%不多啊。那你算过它的收益率比3%高多少吗？高66.67%。


第三件事，是购买任何金融产品，一定要搞清楚它的底层逻辑。比如说，指数基金是由严格审批设立的公募基金公司发行的，被严格监管的，用于购买一篮子指数成分股的基金产品。其募资监管、交易持仓品种，都是透明且规范的。


你知道你的钱被谁拿去了，被拿去投资什么品种了，整个流程是否清晰可控。这都是非常非常重要的。


第四件事，在金融行业，大的机构相对来说要比小机构稳。但更加实质的并不是大小，而是这家机构的经营理念和方向。2008年，大如雷曼兄弟都瞬间轰然倒下。所以你的钱放在哪里，认真研究这家机构的本质就非常重要。从哪里研究？历史、领导人的风格、财务状况、组织构成、股东穿透。

这其实跟绿灯过马路要不要看车差不太多。监管和金融犯罪，本来就是猫鼠游戏。绝大多数情况下，监管可以堵上所有漏洞。但这是一个持续进行的游戏，未来各种金融创新也会层出不穷，谁也不能保证一点问题都不会有。所以，作为普通投资者的我们，在督促监管做得更好的情况下，提高自身能力，能够更好的保护好自己的血汗钱，恐怕也是不能忽略的一个修行方向。

**三个维度（14:55:31）**

以我现在对于投资的理解，有三个点在做配置的时候特别重要。


一个是价值，一个是趋势，一个是景气度。


价值不用多说。


趋势则代表了顺势者昌，逆势者亡。时势造英雄，不识时务者不可称为俊杰。


还有一点是景气度。即行业的中短期发展。这一点与价值的区别在于，价值会在长期得到验证，而景气度则是机构特别关注的中短期因素。


为什么呢，因为一般的机构没有时间去长期体现价值。他们需要月度、季度、年度排名，来确定江湖地位，来确定年终奖金额，来确定提成的多寡。无论这一波景气度过去，这行业这公司是否真的能长期好下去又或者只是昙花一现，要找个炒作的理由，中短期的景气度是特别重要的。
